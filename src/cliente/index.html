Node.JS

Node.js es un entorno de ejecución de JavaScript del lado del servidor que permite a los desarrolladores
construir aplicaciones escalables y de alto rendimiento utilizando JavaScript como lenguaje de programación.

Servidor

Un servidor local con Node.js es un servidor web que se ejecuta en la máquina del usuario, facilitando el
desarrollo y prueba de aplicaciones web utilizando Node.js como entorno de ejecución del lado del servidor.

Relacion con JavaScript

Node.js amplía el uso de JavaScript más allá del entorno del navegador, permitiendo ejecutarlo en el lado del servidor.
Esto posibilita el desarrollo de aplicaciones backend utilizando JavaScript, lo que unifica el lenguaje
de programación a lo largo de toda la pila tecnológica, facilitando la creación de aplicaciones web escalables y eficientes.
